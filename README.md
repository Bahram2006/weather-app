# WeatherNow – Modern Weather Web Application

A modern minimal weather web app that lets you search for a city and view current weather conditions using the OpenWeatherMap API.

## Tech stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- OpenWeatherMap API

## Features

- City search
- Weather card
- Temperature
- Humidity
- Wind speed
- Weather icon

## Setup

Install dependencies:

npm install

Run the project:

npm run dev
